Herramienta
Descripción
Área de un círculo Esta herramienta permite calcular el área de un círculo a partir del diametro o del radio
Área de un Rectángulo Esta herramienta permite calcular el área de un rectángulo o de un cuadrado a partir de su lado mayor y lado menor en el caso de un rectángulo o a partir de un lado en el caso del cuadrado
Área de un Triángulo Esta herramienta permite calcular el área de un triángulo rectángulo de 3 diferentes maneras, la primera conociendo su base y altura, la segunda conociendo dos de sus lados y el angulo entre ellos, la tercera mediante el Teorema de Euclides
Volumen de un Paralelepipedo Esta herramienta permite calcular el volumen de un paralelepipedo conociendo el valor de sus 3 dimensiones, alto, largo y ancho.
Volumen de un Cilindro Esta herramienta permite calcular el volumen de un cilindro a partir del valor de su radio y su altura
Volumen de una Piramide Esta herramienta permite calcular el volumen de una piramide con base cuadrada o rectangular, una piramide triangular y un cono.
